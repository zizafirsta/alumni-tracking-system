# Alumni Tracking System

This project is a web-based system designed to track alumni information from various public sources.

## Features

- Manage alumni data
- Alumni tracking simulation
- Confidence score evaluation
- Dashboard statistics
- Alumni search
- Data visualization using charts

## Technology

- Node.js
- Express.js
- HTML CSS JavaScript
- Bootstrap
- Chart.js

## How to Run

1. Install dependencies

npm install

2. Run server

node server.js

3. Open browser

http://localhost:3000
